<a id="en-english"></a>
## EN English

A lightweight Windows GUI utility to safely backup extensions from various VS Code-based IDEs directly from your local directories.

### 🌟 Features
- **Direct Local Scan**: Reads local `.vscode`, `.cursor`, `.windsurf` directories to avoid CLI errors.
- **Multi-IDE Support**: Supports VS Code, VSCodium, Cursor AI, Windsurf, and Google Antigravity.
- **Responsive UI & Smart Downloads**: Features asynchronous processing and rate-limit protection.

### 📥 Installation & Usage
1. Go to the **[Releases](../../releases/latest)** page and download the latest `.exe` installer.
2. Install and run **Extension Backup Tool**.
3. Select your IDEs and click **Mulai Backup**.

*(Note: If your Antivirus flags the compiled `.exe` as a false positive, you can safely run the raw `script.ps1` from this repository instead).*

## ID Bahasa Indonesia

Utilitas Windows GUI ringan untuk mencadangkan (backup) ekstensi dari berbagai IDE berbasis VS Code langsung dari direktori lokal Anda dengan aman.

### 🌟 Fitur Utama
- **Pemindaian Lokal**: Membaca folder `.vscode`, `.cursor`, `.windsurf` langsung untuk menghindari error CLI.
- **Mendukung Banyak IDE**: Kompatibel dengan VS Code, VSCodium, Cursor AI, Windsurf, & Google Antigravity.
- **UI Responsif & Unduhan Cerdas**: Dilengkapi pemrosesan asinkron dan perlindungan *rate-limit*.

### 📥 Instalasi & Penggunaan
1. Buka halaman **[Releases](../../releases/latest)** lalu unduh installer `.exe` terbaru.
2. Instal dan jalankan **Extension Backup Tool** dari komputer Anda.
3. Pilih IDE Anda dan klik **Mulai Backup**.

*(Catatan: Jika program Antivirus Anda memblokir file `.exe` karena false positive, Anda tetap bisa menjalankan file mentah `script.ps1` dari repositori ini).*
